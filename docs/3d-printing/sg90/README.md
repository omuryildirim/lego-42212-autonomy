# SG90 mini servo adapters

Two printable parts for fitting an SG90 mini servo into a LEGO Technic chassis: a spline adapter that converts the servo's output spline into a standard LEGO axle, and a bracket that clips the servo body onto a stud row.

## Spline adapter (`S90 - 4.8mm - 6.5mm - 3l axle.stl`)

Connects an SG90 mini servo to LEGO pieces.

### Interface details

- Servo side: SG90, 21T spline connection
- Servo spline nominal diameter: 4.8 mm
- Lego side: axle-compatible interface for pieces such as Lego Technic, Axle 6L (#3706)
- Lego axle nominal dimensions: 4.4 mm x 4.4 mm

### Design note

This adapter was inspired by testing Lego Technic, Pin Connector Round 2L with Slot (#62462) as a press-fit option on an SG90 spline. The printed part was created to provide a more stable axle connector.

## Servo bracket (`SG90-bracket.stl`)

LEGO-stud-compatible block that holds the SG90's body. The output spline pokes out one end (where the spline adapter attaches), and the side is open so the 3-pin servo lead exits cleanly. The top has standard LEGO studs, so the bracket clips onto any baseplate, Technic frame, or beam with a matching stud row.

### Interface details

- Servo side: cradle sized to the SG90 body
- Lego side: standard LEGO stud spacing on the top face for chassis mounting
