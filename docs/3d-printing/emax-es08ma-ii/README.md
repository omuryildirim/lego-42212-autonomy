# EMax ES08MA II Mini Servo To Lego Axle Adapter

EMax ES08MA II is a full metal version of the commonly used SG90-size mini servo.

This piece is designed to connect an EMax ES08MA II mini servo to Lego pieces.

## Interface details

- Servo side: EMax ES08MA II, Phi 3.9 15T spline connection
- Servo spline nominal diameter: 3.9 mm
- Lego side: axle-compatible interface for pieces such as Lego Technic, Axle 6L (#3706)
- Lego axle nominal dimensions: 4.4 mm x 4.4 mm
